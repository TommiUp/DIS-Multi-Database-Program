SETUP
1. Create Postgre database: store_ecommerce
2. Insert tables and values to store_ecommerce (demonstration purposes)
3. Create .env at project root with: PG_PASSWORD=your_password
4. Create Mongo database and insert the collections (demonstration purposes)
5. Install all dependencies
6. Run: python DatabaseProgram.py
